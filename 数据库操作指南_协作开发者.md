## 四、图片上传指南（不建议，可参考“常见问题”）

### 4.1 上传图片到云存储

1. 在云开发控制台选择「云存储」
2. 进入对应的文件夹（如 `images/scenic/` 或 `images/heritage/`）
3. 点击「上传文件」
4. 选择本地图片上传
5. 上传成功后，右键点击文件选择「复制文件路径」

### 4.2 图片路径格式

```
cloud://cloud1-7gabd815fd2c236e.636c-cloud1-7gabd815fd2c236e-1418729882/images/scenic/xxx.jpg
```

> ⚠️ **注意**：在数据库中填写时，只需使用 `cloud://` 开头的路径即可。

### 4.3 推荐的存储路径

| 内容类型 | 存储路径 |
|:---|:---|
| 景点封面图 | `images/scenic/` |
| 非遗封面图 | `images/heritage/` |
| 打卡图片 | `images/checkin/` |
| 打卡海报 | `images/poster/` |
| 语音音频 | `images/audio/` |
| 轮播图 | `images/banner/` |

## 五、常见问题

### Q: 图片上传？

**A**: 直接在数据库中上传本地图片即可。

### Q: 经纬度如何获取？

**A**: 
1. 在腾讯地图官网（lbs.qq.com）搜索地点
2. 点击目标地点，查看详情中的坐标
3. 或在地图上右键选择「显示坐标」
4. **也可直接问AI，给是参考都江堰**
