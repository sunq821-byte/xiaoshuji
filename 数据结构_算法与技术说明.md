## 二、算法

### 2.1 Haversine 球面距离公式

**文件**：`utils/mapUtils.js` → `calculateDistance(lat1, lng1, lat2, lng2)`

用于判断用户是否在目标打卡点 500m 范围内。

```
a = sin²(Δlat/2) + cos(lat1) × cos(lat2) × sin²(Δlng/2)
c = 2 × atan2(√a, √(1−a))
distance = R × c        // R = 6 371 000 m
```

**复杂度**：O(1)，固定数学运算。

---

### 2.2 腾讯地图 Polyline 折线解码（Delta 编码）

**文件**：`utils/tencentMapService.js` → `_decodePolyline(encoded)`

腾讯地图路径规划返回压缩后的坐标串，格式为分号分隔的增量值序列，每个值需除以 1,000,000 后与前一点累加，还原出绝对坐标。

```
points = []
lat_sum = 0, lng_sum = 0
for each pair (dLat, dLng) in encoded.split(';'):
    lat_sum += dLat / 1_000_000
    lng_sum += dLng / 1_000_000
    points.push({latitude: lat_sum, longitude: lng_sum})
```

**复杂度**：O(n)，n 为路径点数量。

### 2.3 多字段正则 OR 搜索

**文件**：`utils/cloudCall.js` → `buildSearchCondition(keyword, fields)`  
**使用方**：`heritageList.vue`、`scenicList.vue`、`search.vue`

将关键词编译为云数据库 `RegExp` 对象，对多个字段构造 OR 查询条件，实现全文模糊搜索。

```javascript
const regex = db.RegExp({ regexp: keyword, options: 'i' });
const condition = dbCmd.or(
  fields.map(field => ({ [field]: regex }))
);
// 搜索字段：name、brief、category、location、tags
```

**复杂度**：O(n)（n 为集合文档数），依赖云数据库索引加速。

### 2.4 跳过-限制分页（Skip-Limit Pagination）

**文件**：`cloudCall.js` → `dbQuery()`；各列表页

```
page 1: skip=0,  limit=pageSize
page 2: skip=10, limit=pageSize
hasMore = (返回条数 === pageSize)
```

加载更多时追加数据到列表末尾，`hasMore=false` 时隐藏"加载更多"按钮。

`initViewCount` 云函数中同样使用此算法批量初始化字段（批大小 100，受云开发单次查询上限约束）。

### 2.6 类别去重与排序

**文件**：`heritageList.vue`

从数据库返回的非遗项目中提取类别标签，处理逗号分隔的多类别字段，使用 `Set` 去重后按预设顺序排列。

```javascript
// 步骤1：展平所有 category 值
const rawCategories = list.flatMap(item =>
  String(item.category || '').split(',').map(s => s.trim())
);
// 步骤2：Set 去重
const unique = [...new Set(rawCategories)].filter(Boolean);
// 步骤3：按自定义顺序排序
const ORDER = ['传统音乐', '传统舞蹈', '传统戏剧', ...];
unique.sort((a, b) => ORDER.indexOf(a) - ORDER.indexOf(b));
```

### 2.8 云存储图片批量临时链接转换

**文件**：`utils/cloudCall.js` → `getTempFileURL(fileID)` / `processImageFields(data, fields)`

云存储 `cloud://` 路径在小程序 `<image>` 组件中无法直接显示，需通过 `wx.cloud.getTempFileURL` 转换。

- **批次大小**：每批最多 10 个 fileID（微信 API 限制）。
- **超时保护**：使用 `Promise.race([fetchPromise, timeoutPromise(5000)])` 防止长时间挂起。
- **降级策略**：超时或失败时，原样返回 `cloud://` 路径，避免页面白块。
- **字段类型感知**：自动识别字段值为单个字符串或字符串数组，分别处理。

```
输入：cloud://xxx/images/heritage/cover.jpg
输出：https://cdn-miniapp.xxx.com/images/heritage/cover.jpg?sign=...
```

### 2.11 用户登录状态机

**文件**：`utils/auth.js`、`pages/mine/mine.vue`

三态状态机，单向流转：

```
游客态（无 openid）
    ↓ silentLogin()（wx.login → getOpenId 云函数）
已登录 · 无资料（有 openid，无 nickName/avatarUrl）
    ↓ authorizeLogin()（chooseAvatar + nickname input → user-login 云函数）
已登录 · 有资料（完整用户信息）
```

Session 有效期 7 天，到期后 `ensureLogin()` 自动触发静默重新登录。
