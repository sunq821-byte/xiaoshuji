- **新增方法**：

  - `planSingleSegment()` - 单段路线规划
  - `extractPolylineCoords()` - 提取 polyline 坐标
  - `finishMultiSegmentRoute()` - 完成多段路线绘制
- **额外修复**：移除「公交」出行方式

### 9. 路线规划渲染错误修复
- **文件**：`pages/route/route.vue` 的 `drawRouteOnMap` 方法
- **错误信息**：`MultiPolyline.geometries: paths属性无效` + `Invalid typed array length: -3`
- **原因**：腾讯地图返回的 `polyline` 是**编码格式**：`[lat, lng, dlat, dlng, dlat, dlng, ...]`，需要用增量解码
- **修复方案**：
  1. 修正解码逻辑：前两个是绝对坐标，后续是增量（除以1e6）
  2. 验证坐标值类型和范围（中国区域：lat 15-55, lng 73-136）
  3. 添加 `simplifyRoute` 方法限制坐标点数量（最多500点）
  4. catch 块中清空 `routeLine = []` 防止错误累积
