# 部署文档

## 特别说明

本作品为**微信小程序**，依赖微信云开发环境运行。源代码完整，配合本部署文档可复现完整运行环境。

### 小程序特殊性

与传统软件不同，微信小程序需要：
1. 微信公众平台账号（个人/企业）
2. 微信云开发环境（腾讯云）
3. 第三方API Key（如百度TTS、腾讯地图）

**如果无法完全部署**，请参考以下材料：

- 演示视频（已附）：展示完整功能

## 一、环境要求

### 1.1 必需环境

| 环境 | 版本要求 | 说明 |
|------|----------|------|
| Node.js | ≥ 16.0.0 | 云函数运行环境 |
| 微信开发者工具 | 最新稳定版 | 微信官方IDE |
| 微信开发者工具-增强版 | 最新 | 支持云开发增强特性 |

### 1.2 第三方服务

| 服务 | 用途 | 费用 |
|------|------|------|
| 微信云开发 | 数据库、云函数、存储 | 免费（有限额） |
| 百度智能云（TTS） | 文字转语音 | 免费（基础音库） |
| 腾讯位置服务 | 地图、导航 | 免费（基础配额） |

## 二、微信云开发环境配置

### 2.1 创建云开发环境

**步骤1**：登录微信公众平台（https://mp.weixin.qq.com/）

**步骤2**：注册小程序（个人/企业账号均可）

**步骤3**：下载并安装微信开发者工具

**步骤4**：在微信开发者工具中创建项目，选择「云开发」模板

**步骤5**：创建云开发环境

```
环境名称：生产环境
环境ID：cloud1-xxxxxxx（系统自动生成）
```

⚠️ **记录环境ID**，后续配置需要用到。

### 2.2 获取环境ID

```
微信开发者工具 → 云开发控制台 → 设置 → 环境ID
```

环境ID格式：`cloud1-xxxxxx-xxxxx`

## 三、云函数部署

### 3.1 云函数列表

本项目需要部署以下云函数（共14个，已在源代码包中完整提供）：

| 序号 | 云函数名称 | 功能说明 | 代码状态 |
|------|------------|----------|----------|
| 1 | `getOpenId` | 静默登录，获取用户OpenId | ✅ 已提供 |
| 2 | `user-login` | 用户授权登录 | ✅ 已提供 |
| 3 | `favorite-add` | 添加收藏 | ✅ 已提供 |
| 4 | `favorite-remove` | 取消收藏 | ✅ 已提供 |
| 5 | `favorite-list` | 获取收藏列表（支持筛选） | ✅ 已提供 |
| 6 | `checkin-submit` | 提交打卡 | ✅ 已提供 |
| 7 | `checkLocation` | 打卡位置验证 | ✅ 已提供 |
| 8 | `getMyCheckin` | 获取我的打卡记录 | ✅ 已提供 |
| 9 | `tts-guide` | 百度TTS文字转语音 | ✅ 已提供 |
| 10 | `geo-reverse` | 逆地址解析 | ✅ 已提供 |
| 11 | `getRoutes` | 获取推荐路线 | ✅ 已提供 |
| 12 | `submit-feedback` | 提交反馈 | ✅ 已提供 |
| 13 | `addBrowseHistory` | 添加浏览历史（去重） | ✅ 已提供 |
| 14 | `initViewCount` | 初始化浏览量 | ✅ 已提供 |

### 3.2 部署云函数（方法一：微信开发者工具）

**步骤1**：在微信开发者工具中打开项目

**步骤2**：右键点击 `cloudfunctions` 文件夹

**步骤3**：选择「上传并部署：云端安装依赖」

**步骤4**：等待部署完成（会自动安装 wx-server-sdk 依赖）

**步骤5**：重复步骤2-4，对每个云函数进行部署

> ⚠️ **注意**：请确保每个云函数目录下的 `package.json` 中的 `dependencies` 正确，微信开发者工具会自动读取并安装。

### 3.3 部署云函数（方法二：命令行）

```bash
# 安装微信开发者工具CLI
npm install -g @cloudbase/cli

# 登录
tcb login

# 部署所有云函数
tcb fn deploy --force --env 你的环境ID

# 或单独部署某个云函数
tcb fn deploy tts-guide --force --env 你的环境ID
```

### 3.4 配置云函数环境变量

部分云函数需要配置环境变量（API Key等）：

**操作位置**：微信开发者工具 → 云开发控制台 → 云函数 → 函数配置 → 环境配置

| 云函数 | 变量名 | 说明 |
|--------|--------|------|
| `tts-guide` | `BAIDU_TTS_API_KEY` | 百度TTS API Key |
| `tts-guide` | `BAIDU_TTS_SECRET_KEY` | 百度TTS Secret Key |
| `geo-reverse` | `TENCENT_MAP_KEY` | 腾讯地图 Key（可选，有备用方案） |

## 四、云数据库配置

### 4.1 需要创建的集合

在微信开发者工具 → 云开发控制台 → 数据库，创建以下集合：

| 集合名称 | 说明 | 权限设置 |
|----------|------|----------|
| `heritage` | 非遗项目数据 | 所有用户可读 |
| `scenic` | 景点数据 | 所有用户可读 |
| `swiper` | 轮播图数据 | 所有用户可读 |
| `routes` | 推荐路线数据 | 所有用户可读 |
| `user_favorites` | 用户收藏记录 | 仅创建者可读写 |
| `checkin` | 打卡记录 | 仅创建者可读写 |
| `browse_history` | 浏览历史 | 仅创建者可读写 |
| `feedback` | 用户反馈 | 所有用户可写，仅创建者可读 |
| `users` | 用户信息 | 仅本人可读写 |
| `tts_cache` | TTS缓存（自动创建） | 所有用户可读写 |

## 五、第三方API申请

### 5.1 腾讯地图Key申请

**步骤1**：访问腾讯位置服务官网
https://lbs.qq.com/

**步骤2**：注册/登录腾讯账号

**步骤3**：进入「控制台」→「应用管理」→「我的应用」

**步骤4**：点击「创建应用」，填写应用信息

**步骤5**：添加「Key」→ 选择「微信小程序」

**步骤6**：配置服务器域名白名单（如需要）

**步骤7**：复制Key备用

```
Key格式：NV7BZ-XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX
```

### 5.2 百度TTS申请

**步骤1**：访问百度智能云
https://ai.baidu.com/

**步骤2**：注册/登录百度账号

**步骤3**：进入「语音技术」→「短文本在线合成」

**步骤4**：领取免费资源（基础音库有免费额度）

**步骤5**：创建应用，获取 API Key 和 Secret Key

**步骤6**：复制备用

```
API Key格式：QjXwlS2aaBXK2Mv1e5C8KAoC
Secret Key格式：BgSc0CyElAvHfNFxetp5FoPvxnF1PGUV
```

**步骤1**：登录微信公众平台

## 六、项目配置

### 6.1 修改环境ID

打开 `config.js`，修改云环境ID：

```javascript
// config.js 第11行
export const CLOUD_CONFIG = {
  // 云开发环境ID（修改为你的环境ID）
  env: 'cloud1-xxxxxxx-xxxxx',  // ← 替换为你的环境ID
  enabled: true
}
```

### 6.2 配置腾讯地图Key

```javascript
// config.js 第19行
export const MAP_CONFIG = {
  // 腾讯地图 Key（修改为你的Key）
  key: '你的腾讯地图Key',  // ← 替换为你的Key
  // ...
}
```

### 6.3 配置百度TTS

**方式一：修改配置文件**

```javascript
// config.js 第33-34行
export const TTS_CONFIG = {
  apiKey: '你的百度API Key',      // ← 替换
  secretKey: '你的百度Secret Key', // ← 替换
  // ...
}
```

**方式二：云函数环境变量（推荐，更安全）**

在云函数 `tts-guide` 的配置中设置环境变量：
- `BAIDU_TTS_API_KEY`
- `BAIDU_TTS_SECRET_KEY`

然后修改 `cloudfunctions/tts-guide/index.js`：

```javascript
// 移除硬编码，优先使用环境变量
const API_KEY = process.env.BAIDU_TTS_API_KEY;
const SECRET_KEY = process.env.BAIDU_TTS_SECRET_KEY;
```

### 7.3 常见问题

| 问题 | 解决方案 |
|------|----------|
| 云函数调用失败 | 检查云开发环境是否开通 |
